# Data Structures

## Arrays

- stores the same type of data.
- Can imagine like a block of data
- Contiguous in memory
- Linear
- Index starts from 0
- Non-primitive
- Pass by reference

Syntext in C++<br>
<pre>  
  data_type var_name[size_of_array] = {Value1, Value2,...,Value_size_of_array};
  data_type var_name[] = {Value1, Value2,...};
</pre>

# Algorithms

## Linear Search

- Visit each node and compare your target value.<br>
--> Implement a loop and compare your target value with each value.<br>
--> When you find the value, break the loop.<br>
--> If the value does not exist, then we can print a message or return -1.

**Time Complexity: O(n)**
